# Machine Learning Regression

This project explores Linear and Multiple Regression models to predict CO2 emissions based on vehicle features.

## Project Structure

- **Linear_Reg**: Contains `linearReg.ipynb`. It uses the `FuelConsumptionCo2.csv` dataset to explore the relationship between a single feature (like engine size) and CO2 emissions.
- **Multi_Reg**: Contains `multipleReg.ipynb`. It extends the analysis by using multiple features (Engine Size, Cylinders, Fuel Consumption) to provide a more accurate prediction of CO2 emissions.

## Dataset

The dataset contains model-specific fuel consumption ratings and estimated carbon dioxide emissions for new light-duty vehicles for retail sale in Canada.

## Key Concepts

- Exploratory Data Analysis (EDA)
- Model evaluation metrics (MAE, MSE, RMSE, R2-score)
- Visualizing regression results
