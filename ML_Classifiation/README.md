# Machine Learning Classification

Implementation of various classification algorithms and techniques.

## Project Structure

- **DecisionTree**: Contains `DT.ipynb` which demonstrates the use of `DecisionTreeClassifier` on a drug dataset (`drug200.csv`) to predict which drug might be appropriate for a patient based on Age, Sex, BP, and Cholesterol.
- **KMean**: Contains `KMeanLab.ipynb` focusing on clustering techniques.
- **KNN**: Includes both a from-scratch implementation and an sklearn-based lab to explore K-Nearest Neighbors.
- **LogisticRegression**: Features `logisticRegressionLab.ipynb` and `MultiClassPredic.ipynb` for binary and multiclass classification tasks.

## Getting Started

Each directory contains a Jupyter notebook that explains the implementation details and provides visualizations for the respective algorithm.
