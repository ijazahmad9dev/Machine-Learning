# Machine Learning Projects

A collection of machine learning projects focusing on various algorithms, from classification to regression.

## Overview

This repository contains several sub-projects, each exploring different aspects of machine learning using datasets like credit card transactions, NYC taxi records, and fuel consumption ratings.

### Included Projects

1.  **[Machine Learning Classification](ML_Classifiation/README.md)**: Implementations of KNN, K-Means, Decision Trees, and Logistic Regression.
2.  **[Credit Card Fraud Detection](ML_CreditCard_FruadDetection/README.md)**: Binary classification to identify fraudulent transactions using accelerated training with Snap ML.
3.  **[Machine Learning Regression](ML_Regression/README.md)**: Predicting CO2 emissions using Linear and Multiple Regression.
4.  **[Taxi Tip Prediction](Taxi-Tip-Prediction/README.md)**: A regression task using DecisionTreeRegressor to predict taxi tips from large-scale trip data.
5.  **[Multimodal Meme Sentiment Analysis](Multimodal_Meme_Sentiment_Analysis/README.md)**: A multimodal sentiment classifier for Internet memes using text and image ensembling with a Flask web interface.

## Technologies Used

- Python
- Jupyter Notebooks
- Scikit-Learn
- Snap ML (IBM)
- Pandas, NumPy, Matplotlib, Seaborn

## Getting Started

Each sub-project directory contains its own README with specific details about the dataset, models implemented, and how to run the notebooks.
